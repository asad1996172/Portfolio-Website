# Portfolio-Website
Flask based portfolio web application. It contains the following sections:

* About
* Publications
* Projects
* Experience
* Skills
* Papers Bibliography 
* Education

All of these sections have their own .json file which any one can edit to create their own portfolio website. Papers Bibliography doesn't have a json file. Instead it has a google sheets file which is downloaded and then shown on the portfolio website.

Feel free to use this template to create your own portfolio website.
